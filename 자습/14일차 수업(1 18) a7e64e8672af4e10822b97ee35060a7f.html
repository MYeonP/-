14일차 수업(1.18)

자습

  • date
    • 날짜, 시간, 요일 보기
  • cal
    • 캘린더
  • cal 년도 (ex : cal 2023)
    • 해당 년도 달력 표시
    • cal 5 1990 : 1990년 5월 달력 표시

  • who
    • 접속자 확인
  • w
    • 좀 더 자세한 로그인 정보
  • arch
    • architecture 약자 : CPU 정보 나타냄
  • logname
    • 로그인 한 사람 정보(나)

  • id
    • id의 고유 넘버 표시
    • uid=1000(j) gid=1000(j) 그룹들=1000(j),4(adm),24(cdrom),27(sudo),30(dip),46(plugdev),110(lxd)
    • id j(나) : 나의 정보
    • id 다른 아이디 (ex : id root / id ocn) : 해당 아이디의 고유 넘버
  • id -u j / id -u ocn
    • 깔끔하게 아이디 숫자만 보여줌
  • id -g / id -g root / id -g ocn
    • 해당 아이디의 그룹 숫자 보여줌

  • ctrl + l
    • 클리어
  • hostname --help
    • hostname 옵션 정보 확인(도움말)
  • history
    • 이용한 명령어 순서대로 볼 수 있음
  • hostname
    • 내 이름
  • uname
    • 리눅스 이름

  • 명령 프롬프트

  • . / .. (점 하나, 점 둘)
    • .(점 하나) : 자기 자신
    • ..(점 둘) : 부모 디렉토리
    • ex : cd . : 자기 자신(이동 없음)
    • ex : cd . . : 부모 디렉토리로 이동(d1 → aaa)
    • ex : cd d1 : d1 디렉토리로 이동(aaa → d1)

  • mkdir c:\aaa
    • aaa라는 디렉토리 생성
    • cls : clear space / 도스 화면 밀기
    • C:\Users\bitcamp > cd c:\aaa(c 드라이브의 aaa 디렉토리로 접속)
    • c:\aaa>
    • mkdir d1 d2 d3 : d1 d2 d3 디렉토리 생성

  • mkdir ddd
    • ddd 라는 디렉토리 생성
  • touch bbb
    • bbb라는 빈 파일 생성
  • ls : list 약자
    • ls : 디렉토리 안의 파일 목록 보여줌
    • ls -l : 디렉토리 안의 파일 목록을 자세하게 보여줌
    • ls -a : 숨김 파일 포함 보여줌
    • ls -al : 자세하게 보여줌 (all)
    • ls *.txt : txt 확장자 파일을 다 보여줌
    • ex) -rw-rw-r-- 1 j j 0 1월 18 20:05 bbb / drwxrwxr-x 2 j j 4096 1월 18 20:05 ddd
      • 처음 -로 시작하는 것 : 파일(file)
      • 처음 d로 시작하는 것 : 디렉토리(file)

  • pwd
    • print working directory : 현재 위치
  • rmdir ddd
    • ddd 라는 디렉토리 삭제

  • rm -rf ddd
    • ddd라는 디렉토리 강제 삭제
      • ex : ddd 내부에 디렉토리가 생성 되어있을 때 rmdir로는 삭제가 되지 않지만 rm -rf ddd로는 삭제 가능
  • 응용하기
    • mkdir d{1..5} : 한꺼번에 d1~d5 디렉토리 생성
    • mkdir k{1..5} : 한꺼번에 k1~k5 디렉토리 생성
    • rmdir d{1..5} : 한꺼번에 d1~d5 디렉토리 삭제
    • rmdir k{1..5} : 한꺼번에 k1~k5 디렉토리 삭제

  • 경로 표시 방법 (vi) 모드
    • root로 전환(관리자 모드)
    • vi 모드에서 앞쪽 넘버 표시 하는 방법
      • Esc 키를 눌러 명령 모드로 전환
      • : (colon)을 누르면 커서가 화면 왼쪽 하단 모서리로 이동. set number 또는 set nu를 입력하고 Enter

  • 내 home으로 돌아오기
    • cd로 경로 이동 후 한번에 돌아오기 : cd ~ 혹은 cd
